# babydoc
育兒資料的整理

## HOME
* EAT
1. 消毒鍋(消毒+烘乾)
2. 奶瓶：寬口、玻璃(到足量水+奶粉)
3. 奶瓶刷、奶嘴刷、專用洗劑

* 用
1. 問月子中心尿布size
2. 濕紙巾，屁屁膏(凡士林、理膚膏水B5)
3. 監視器(3~4個月再買)

* 穿
1. 紗布衣(3~5件)，紗布巾
2. 包屁衣(4~7件)
3. 肚圍(1~2件)
4. 帽子、襪子
5. 專用洗衣精

* 洗澡
1. 澡盆
2. 專用沐浴乳

* 睡
1. 嬰兒床(70cm↓，至少同房間一年)
2. 蚊帳
3. 浴巾(4-8件),床單(1-2件)，乳膠床墊)(60x120cm)
`king size: 136*77 cm`
4. 床圍
5. 房踹睡袍
6. 音樂鈴 or 黑白書

* 外出
1. 推車
2. 汽座
3. 背巾
